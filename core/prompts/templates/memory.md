# 记忆快照

> **重要说明**: 此文件为会话启动时加载的记忆快照，会话过程中不会自动更新。
> 如需修改记忆，请使用 memory 工具显式操作。

## 记忆索引

此处包含会话启动时加载的记忆索引内容。记忆索引用于快速检索相关记忆，
不包含完整的记忆内容。

## 使用说明

- 记忆快照在每次会话启动时加载一次
- 会话过程中产生的新信息需要通过 memory 工具写入
- 如需回忆特定记忆，使用 memory(operation='search', query='关键词') 搜索
- 如需读取完整记忆，使用 memory(operation='read', title='记忆标题')
